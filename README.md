# Social App (React)

## 📌 Overview
This project is a beginner-level social media application built with React.
The main goal of the project was to practice core React concepts by building a real multi-page application instead of isolated components or demos.

The app focuses on routing, authentication flow, reusable components, and handling API requests in a structured way.

---

## 🚀 Features
- User authentication (login & signup flow)
- Protected routes (authenticated vs unauthenticated pages)
- Home feed with posts
- Post details page
- User profile page
- Create and delete posts
- Toast notifications for user feedback
- Not found (404) page handling

---

## 🧭 Pages
- Home
- Login
- Signup
- Profile
- Post Details
- Not Found (404)

## 🛠️ Tech Stack

## 🗂️ Project Structure
src/
├── assets/        # Images and static files
├── components/    # Reusable UI components
├── context/       # Global state using Context API
├── hooks/         # Custom React hooks
├── pages/         # Application pages (routes)
├── services/      # API and request logic
├── App.jsx        # Main app component
└── main.jsx       # Application entry point

---

## 🎯 Purpose of the Project
This project was built as a learning project to:
- Understand how a real React application is structured
- Practice routing and protected routes
- Learn how to manage shared state using Context API
- Apply form validation and API handling
- Improve component organization and readability

It is not intended to be production-ready, but it represents a solid foundation in React fundamentals.

---

## ▶️ Getting Started

Install dependencies:
npm install

Run the project:
npm run dev
